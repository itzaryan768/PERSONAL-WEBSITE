Available Endpoints

Use the following endpoints to interact with the API service:

  • Copilot: /api/copilot?prompt=your_prompt
  • SDXL: /api/sdxl?prompt=your_prompt&model=your_model
  • Wallpaper: /api/wallpaper?query=your_query&number=number_of_images
  • Prompt Generator: /api/promptgen?prompt=your_prompt
  • Song Info v2: /api/songinfo/v2?id=your_song_id
  • Video Info: /api/videoinfo?query=your_query
  • Movie Info: /api/movieinfo?title=your_movie_title
  • DALL-E: /api/dalle?prompt=your_prompt&amount=number_of_images
  • Llama: /api/llama?prompt=your_prompt
  • Mixtral: /api/mixtral?prompt=your_prompt
  • Pinterest: /api/pinterest?query=your_query&limits=number_of_results
  • Movie Info v2: /api/movieinfo/v2?id=your_movie_id
  • Gemini: /api/gemini?prompt=your_prompt
  • Image Generator: /api/gen?q=your_query
  • Facebook Stalker: /api/fbstalk?uid=your_user_id&token=your_access_token
  • Pexels: /api/pexels?query=your_query&number=number_of_images
  • 4K Upscaler: /api/4k?url=your_image_url
  • SIM: /api/sim?chat=your_chat&lang=your_language
  • Lyrics: /api/lyrics?songName=your_song_name
